# Normas premium

## NORMA LEGACY-INVENTARIO-04
Todo módulo o carpeta legacy del repositorio debe estar inventariado con estado y acción recomendada.
No se permite dejar legacy sin trazabilidad documental.
Cualquier borrado futuro de legacy debe referenciar explícitamente `docs/audit/legacy_repo_inventory_final.md`.

## Referencia operativa
Para el cierre de storage legacy por proyecto, usar `docs/operacion/cierre_storage_legacy.md` como guía de ejecución.
